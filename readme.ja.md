## 概要
ROBOMECH 2017におけるポスター「物理シミュレーションと透視投影カメラモデルを用いたバラ積みシーン点群生成プログラムの開発」(2A2-O09)で発表したプログラムです．
三次元点群を用いたバラ積みピッキングのための物体認識・位置姿勢推定アルゴリズムの開発の補助のために作成しました．

本アプリケーションはバラ積みシーン生成プログラム(BinSceneMaker)と，片面点群生成プログラム(VisibleSceneMaker)から構成されます．
三次元点群処理に[Point Cloud Library(PCL)](http://pointclouds.org/)，物理シミュレーションに[Bullet Physics](http://bulletphysics.org/wordpress/)を利用しています．
